## 📋 Descripción del Proyecto

MAXCOMGAMES es una plataforma de e-commerce moderna desarrollada como proyecto terminal, especializada en la venta de productos gaming y tecnología. El proyecto integra tecnologías web modernas para ofrecer una experiencia de usuario fluida y profesional.

## ✨ Características Principales

- 🛒 **Carrito de Compras Interactivo** - Sistema completo de gestión de productos
- 🔍 **Búsqueda y Filtros Avanzados** - Encuentra productos específicos fácilmente
- 📱 **Diseño Responsive** - Experiencia optimizada en todos los dispositivos
- 🚀 **Interfaz Moderna** - UI/UX contemporáneo y atractivo
- ⚡ **Rendimiento Optimizado** - Carga rápida y navegación fluida
- 🔐 **Gestión de Usuarios** - Sistema de autenticación y perfiles

## 🛠️ Tecnologías Utilizadas

## 🚀 Instalación y Configuración

### Prerrequisitos
- Node.js (v14 o superior)
- npm o yarn

### Pasos de Instalación

1. **Clonar el repositorio**
   ```bash
   git clone https://github.com/maxitodev/MaxcomGames.git
   cd MaxcomGames
   ```

2. **Instalar dependencias**
   ```bash
   # Para el frontend
   cd frontend
   npm install
   ```

3. **Ejecutar en modo desarrollo**
   ```bash
   npm start
   ```

4. **Acceder a la aplicación**
   - Abre [http://localhost:3000](http://localhost:3000) en tu navegador

## 📦 Scripts Disponibles

| Comando | Descripción |
|---------|-------------|
| `npm start` | Ejecuta la aplicación en modo desarrollo |
| `npm test` | Lanza los tests en modo interactivo |
| `npm run build` | Construye la aplicación para producción |
